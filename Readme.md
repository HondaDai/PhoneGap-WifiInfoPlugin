PhoneGap WifiInfoPlugin
==============

* License - The MIT License
* Test on cordova 3.4.0


Install Step (Cordova CLI)
--------------------------

### 
	cordova plugin install https://github.com/ManRueda/org.apache.cordova.wifiinfo

API
-----

### Example 1
	window.wifi.info;

### Example 2
	window.wifi.refresh();
